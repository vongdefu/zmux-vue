### 日程管理

- 整个页面需要重新设计：
  - 页面分成三大部分：
    - 页面布局为： 1:1:8，功能区分别为：slogan + 年份标签页 + 任务详情 ；
    - slogan 区： 文字滚动条 ，滚动内容为个人的slogan，暂时展示为： 「看五年，想三年，踏踏实实干一年」，加上一些美化效果；
    - 年份标签页 区：分为两部分，年份标签（显示格式为： YYYY年 ，显示最近五年） 和 年报 按钮；
      - 点击年份按钮： 任务详情 区会展示当年的任务详情； 默认显示当前年份；
      - 点击年报按钮： 会展示 年报 页「页面先做出来，内容先暂时留白，后续补充」；
    - 任务详情页 区：
      - 展示为列表，列表元素为「第 n 周（06/29-07/05）」 右边有一个小按钮；按钮为加号，点击加号进行展开，按钮变成减号，点击减号进行收起，按钮又变成加号；
      - 自动生成当前周以前的列表（日期最早为今年的一月一号），并且按照倒序排列，即最近一周作为第一个节点，并且会在每周一的00:00自动生成新的一周；
      - 自动生成新的一周的时候，旧数据不再允许修改；未完成的任务会连同其根任务自动更新到新的一周；
      - 任务列表的右上方会增加一个「周报」按钮，点击后，任务列表部分变化成周报详情页，格式分为： 已完成任务 + 未完成任务 + 总结；已完成任务和未完成任务按照当前周的任务列表进行汇总，保持树形；总结可以是文字编辑区，由用户自行补充；用户补充时，没两秒自动保存一下，并提示「已保存」；
      - 展开后是任务详情或日程详情，展示为树形，可以添加子任务列表；任务完成之后可以，会展示删除线，删除线为波浪线；右滑弹出删除按钮，类似于苹果手机风格；

日程管理模块中：

- 自动生成 最近五年 的周任务列表（按照周倒序生成，周数以周一为起始时间，最后一个应该是第一周，第一周的开始时间为当年的第一个周一，最后一周的开始时间只要在本年度内都算是本年度的周数，比如2022年最后一周跨年了，但是只要最后一周的周一在本年度内，就算做是本年度的周数）；最近一年的由于未到年底，因此只需要生成到最近的周数即可。
- 每个周数都可以展开，有没有日程都可以进行查看，不能锁定，之前说的锁定是指不能再次编辑，并不是不能查看。
- 年份自动选中为当前年份；
